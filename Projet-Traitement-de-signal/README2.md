# Projet Traitement de signal

This folder contains a signal processing project based on MF-TDMA signal demodulation.

## Overview

The project consists in extracting user slots from a received signal, performing coherent demodulation, applying low-pass filtering, and decoding the transmitted messages.

It also studies the influence of additive white Gaussian noise for different SNR values.

## Main topics

- MF-TDMA
- slot extraction
- AM coherent demodulation
- low-pass filtering
- message decoding
- noise analysis

## Goal

The objective is to recover user messages from a received multiplexed signal and analyze system robustness in the presence of noise.

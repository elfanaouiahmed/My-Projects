# Projet Java

This folder gathers Java projects developed in the context of object-oriented programming.

## Included projects

### Jeu des 13 allumettes
This project implements the logic of the 13 matches game, with different strategies, players, and tests.

### Projet Cercle
This project focuses on the implementation and testing of geometric classes such as points and circles.

## Repository contents

- Java source files
- test files
- project-related documents

## Purpose

The aim of this folder is to organize Java projects covering both algorithmic game logic and object-oriented modeling.
